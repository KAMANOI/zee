# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.

Full text and reporting guidelines:
→ https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you experience or witness behavior that violates this Code of Conduct
in this project's spaces (issues, pull requests, discussions, related
chats), please contact the maintainer privately:

- Email: `hirokikamanoi@gmail.com` with the subject prefix `[zee-conduct]`
- Or open a private GitHub Security Advisory if you prefer that channel.

Reports are handled confidentially. The maintainer will acknowledge
within 7 days and respond on next steps within 14 days.
